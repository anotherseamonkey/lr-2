# Вывод:
В результате лабораторной работы было доказано, что можно уменьшить время выполнения перемножения матриц с помощью распараллеливания программы. Причем из графиков видно, что для матриц, у которых N <= 400, время перемножения почти не менятеся от количества потоков. Для отсальных время выполнения программы значительно сокращается. Для матриц большего размера, чем больше количество потоков, тем быстрее выполняются вычисления, но на графике видно, что в какой-то момент скорость вычислений почти не меняется, несмотря на увеличение числа потоков.
